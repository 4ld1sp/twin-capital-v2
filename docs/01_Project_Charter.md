# Project Charter: Twin Capital Command Center

**Tanggal Dokumen:** April 2026  
**Status Proyek:** Aktif / Eksekusi (Fase 5)  

## 1. Identitas Proyek
- **Nama Proyek:** Twin Capital Command Center – Hybrid AI Auto-Trading Engine
- **Sponsor Proyek:** Aldis (Founder & Chief Architect)
- **Manajer Proyek:** Lead Developer Engineer
- **Anggaran Kasar:** *Internal Self-Funded* (Alokasi anggaran bulanan difokuskan pada biaya *cloud hosting*, Minimax/OpenAI API Tokens, dan Likuiditas Trading).

## 2. Tujuan Utama Proyek (Objectives)
Membangun dan mendeploy terminal eksekutif tersentralisasi yang mampu melakukan pemantauan pasar secara *real-time* dan menjalankan eksekusi *trading* otomatis secara otonom (tanpa emosi manusia). Sistem harus menggabungkan kecerdasan analisis tren (AI Signal Engine) dengan kebrutalan perlindungan aset (Hardcoded Risk Engine).

## 3. Scope & Ruang Lingkup (Deliverables)
✅ **Dalam Cakupan (In-Scope):**
- *Dashboard* berbasis antarmuka grafis kaca premium (*Glassmorphism*).
- Manajemen API Key Terenkripsi standar militer (AES-256-GCM).
- *Dual-Loop* arsitektur bot yang menangani analisis data dan pengaman likuiditas (Drawdown/Trailing Stop).
- Otomatisasi sinkronisasi portofolio antara sistem *trading* lokal dengan bursa Bybit.

❌ **Di Luar Cakupan (Out-of-Scope):**
- Pembuatan dan pembangunan pertukaran mata uang kripto internal (*Own Exchange*).
- Integrasi ke *wallet* desentralisasi (MetaMask) pada fase rilis saat ini.

## 4. Key Stakeholders (Pemangku Kepentingan Utama)
1. **Sponsor / Executive Board:** Memberikan pendanaan dan persetujuan arah produk.
2. **Lead Engineer / Tech Team:** Mengembangkan basis kode *Frontend* dan *Backend*.
3. **Data/Quant Analyst:** Menguji performa *prompt* AI dan memastikan algoritma tidak mengalami *bleeding* dana.

## 5. Kewenangan Resmi
Dengan dokumen ini, tim pengembang diberikan otorisasi penuh untuk menggunakan sumber daya komputasi internal, memodifikasi lingkungan produksi (*Production Server*), dan menghubungkan sistem ini langsung dengan aset ril yang berada di bursa rekanan.
